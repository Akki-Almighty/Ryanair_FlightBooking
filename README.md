## Installation
### Prerequisites
- JDK (Java Development Kit)
- Maven

### Steps
1. Clone the repository:
    ```sh
    git clone https://github.com/Akki-Almighty/Ryanair_FlightBooking.git
    ```
2. Navigate to the project directory:
    ```sh
    cd flight_booking
    ```
3. Install the dependencies:
    ```sh
    mvn clean install
    ```
